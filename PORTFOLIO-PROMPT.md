### 2.3 التفاصيل التقنية

#### الألوان
- **الخلفية:** #F5F3EE (فاتح) / #1A1A1A (داكن)
- **النص الرئيسي:** #0E1C3D (كحلي غامق)
- **النص الثانوي:** #8A8F9B (رمادي)
- **الأكcent:** #C9A84C (ذهبي) — للـ eyebrow فقط
- **زر الثيم النشط:** خلفية #0E1C3D + نص أبيض
- **زر الثيم غير النشط:** إطار رمادي + نص رمادي
- **زر زيارة الموقع:** خلفية #0E1C3D + نص أبيض + شكل دائري (border-radius: 50px)

#### الخطوط
- **العنوان الرئيسي:** Inter/Almarai Bold, حجم كبير (clamp بين 2rem و 3.5rem)
- **الـ eyebrow:** Inter/Almarai Semibold, حجم صغير (var(--text-xs)), لون ذهبي, letter-spacing: 0.15em, text-transform: uppercase
- **الوصف:** Inter/Almarai Regular, حجم متوسط (var(--text-base)), لون رمادي, سطر 1.65
- **الأزرار:** Inter/Almarai Semibold, حجم صغير (var(--text-sm))

#### المسافات
- **بين الـ eyebrow والعنوان:** var(--space-sm) ≈ 10px
- **بين العنوان والوصف:** var(--space-sm) ≈ 10px
- **بين الوصف وأزرار الثيم:** var(--space-lg) ≈ 24px
- **بين أزرار الثيم وزر زيارة الموقع:** var(--space-md) ≈ 16px
- **بين العمودين (معلومات + موكبات):** var(--space-lg) ≈ 24px

#### الموكبات
- **Desktop:** عرض كبير (flex: 1), aspect-ratio: 1440/900, border-radius: 12px
- **Mobile:** عرض ثابت (200px), aspect-ratio: 390/844, border-radius: 12px
- **الترتيب بالعربي:** Mobile يسار + Desktop يمين
- **الترتيب بالإنجليزي:** Desktop يسار + Mobile يمين
- **ظل الموكبات:** 0 8px 32px rgba(0, 0, 0, 0.15)
- **خلفية الموكبات:** نفس لون الخلفية الأساسي

#### أزرار الثيم
- **الشكل:** pill shape (border-radius: 50px)
- **الحجم:** padding: 6px 14px
- **التنسيق:** inline-flex, gap: 2px, خلفية #ECEAE4
- **الزر النشط:** خلفية #C9A84C (ذهبي) + نص #0E1C3D
- **الزر غير النشط:** شفاف + نص #8A8F9B

#### زر زيارة الموقع
- **الشكل:** دائري (border-radius: 50px)
- **الحجم:** padding: 14px 32px
- **الخلفية:** #0E1C3D
- **النص:** أبيض, font-weight: 600
- **النقطة الخضراء:** 8px دائرة خضراء (#22C55E) نابضة (animation: pulse-dot)
- **السهم:** → (يسار لليمين في LTR, يمين لليسار في RTL)

## 3. سلوك التفاعل

### 3.1 تبديل الثيم (Theme Toggle)
- عند الضغط على "الوضع الداكن" / "Dark Mode":
  - الموكبات تتحول لخلفية داكنة (#1A1A1A)
  - الزر يصبح نشط (خلفية ذهبية)
- عند الضغط على "الوضع الفاتح" / "Light Mode":
  - الموكبات ترجع للفاتح (#F5F3EE)
  - الزر يصبح نشط

### 3.2 أنيميشن الدخول (GSAP)
- العنوان يظهر بـ clip-path inset
- الموكبات تظهر من الأسفل (y: 40 → 0)
- المعلومات تظهر من الأسفل (y: 40 → 0)
- التأخير: 0.15s بين كل عنصر

### 3.3 Hover Effects
- **الموكبات:** hover يرفعها قليلاً (transform: translateY(-4px))
- **زر زيارة الموقع:** hover يزود المسافة بين النقطة والسهم

## 4. التجاوب (Responsive)

### 4.1 تابلت (≤1024px)
- Grid يتحول لعمود واحد (grid-template-columns: 1fr)
- الموكبات تظهر فوق المعلومات

### 4.2 موبايل (≤768px)
- الموكبات تنقص عرضها (Mobile: 160px)
- الموكبات تتحول لعمود (flex-direction: column)

### 4.3 موبايل صغير (≤480px)
- Mobile mockup: 120px عرض
- تصغير الأزرار قليلاً

---

## 5. ملاحظات مهمة

1. **لا توجد تاغات لغات برمجية** — لا HTML5, لا CSS3, لا JavaScript, لا Responsive
2. **الترجمة تتم عبر data-i18n** — كل نص له مفتاح في i18n.js
3. **الاتجاه يتغير تلقائياً** — RTL للعربي, LTR للإنجليزي
4. **الموكبات حالياً placeholders** — صور حقيقية تُضاف لاحقاً
5. **التصميم مطابق للصور** — الترتيب والألوان والمسافات كما هو موضح
